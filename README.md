# Giovanni Tonucci Portfolio

A single-page portfolio for Giovanni Tonucci, built with React, Vite, Tailwind CSS, and Formspree.

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Formspree contact form
- ESLint

## Available Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Project Structure

- `src/App.jsx` composes the page sections.
- `src/components/` contains the portfolio sections and reusable UI pieces.
- `src/hooks/useScrollFade.js` handles reveal-on-scroll behavior.
- `public/` contains static browser assets such as the favicon.

## Notes

The contact form uses the Formspree form id configured in `src/components/Contact.jsx`.

The CV link in the About section points to the shared Google Drive file for viewing and downloading.
